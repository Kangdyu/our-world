# Our World

[결과물](https://kangdyu.github.io/our-world)

### 기술 스택

- React
- Webpack
- TypeScript
- Redux
- redux-saga
- react-hook-form
- styled-components

### 사용법

- 개발 모드 (IE 11 불가능)
```
npm install
npm start
// 이후 localhost:3000 으로 접속
```

- 빌드
```
npm install
npm run build
// 이후 로컬 서버를 열어 접속
```
